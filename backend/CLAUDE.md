## Project Overview

SEO Report System generates 19-slide PowerPoint reports from Google Search Console, GA4 analytics, and PageSpeed Insights data. It uses AI (LLM) to generate narrative insights and matplotlib for chart visualization.

## Key Commands

```bash
# Run the dashboard (primary interface)
streamlit run app.py

# Generate report programmatically
python -c "from report_generator.pipeline import ReportPipeline; ReportPipeline('sigmainfo').run()"

# Install dependencies
pip install -r requirements.txt
```

## Architecture: 7-Step Pipeline

`ReportPipeline` in `report_generator/pipeline.py` orchestrates the entire workflow:

1. **GSC Data** (`collectors/gsc_service.py`) - Google Search Console: traffic, keywords, pages
2. **GA4 Data** (`collectors/ga4_service.py`) - Channel comparisons (Google, ChatGPT, Bing, etc.)
3. **Core Web Vitals** (`collectors/pagespeed_service.py`) - PageSpeed Insights: FCP, LCP, TBT, CLS, Speed Index
4. **Build Data** (`report_generator/data_builder.py`) - Aggregates into structured JSON
5. **Charts** (`report_generator/chart_generator.py`) - Generates PNG charts via matplotlib
6. **AI Insights** (`ai/insight_generator.py`) - LLM generates narrative text
7. **PPTX** (`report_generator/pptx_generator.py`) - Builds PowerPoint slides

## Adding a Company

Edit `company_config.json`. Required fields:
- `url`: GSC property URL (use `sc-domain:domain.com` for domain properties)
- `ga4_property_id`: Numeric GA4 property ID (empty string if unavailable)
- `brand_terms`: Array of keywords to identify branded searches

## Authentication

- `client_secret.json`: Google OAuth credentials (required for GSC/GA4)
- `token.pickle`: Auto-generated after first OAuth flow
- If OAuth fails, delete `token.pickle` and re-run to re-authenticate
- `GOOGLE_API_KEY` in `.env`: Required for PageSpeed Insights API
